# Pizzola :pizza:

![pizzola]("./logo.svg")

Este es el backend usado para pizzola.

### Dependencias
 
- Nodejs
- NPM

## Ejecutar el servidor

- npm start

## Rutas de expressjs

GET `/menu`

Para obtener todos los items (pizzas) del menú.

POST `/menu`

Para publicar un item (pizza) en el menú

DELETE `/menu/:id`

Para eleminar un item (pizza) del menú.
